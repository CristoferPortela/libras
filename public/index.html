O que é o SignScript?

?

A pergunta que não quer calar... O que é SignScript? E a resposta é bem lógica: Atualmente é apenas uma ideia. Como assim? O SignScript, ainda não existe, mas irá, eu mesmo pretendo criá-lo como um projeto de TCC... mas chega de falar isso, vamos ver alguns pontos sobre ele:

Tem a ver com linguagem de sinais...
Como o própio nome já diz, Sign, que significa sinal, ele terá como base a linguagem de sinais, embora eu pretenda usar sinais que sejam mais universais, apenas poucos são, então terá como maior base a linguagem brasileira de sinais.
Tem a ver com programação e desenvolvimento
Script, embora possa significar varias coisas, todas tem relação com a escrita, ou um papel a se desempenhar, mas não só isso, a palavra Script se popularizou por causa da linguagem JavaScript e muitas bibliotecas, e frameworks baseados nela tem usado essa palavra na sua composção, não indo longe disso, juntei Sign e Script e já podemos imaginar o que irá sair, não é mesmo?
Provavelmente você pensou errado
SignScript não será uma linguagem de programação, isso seria um absurdo, mas tem a ver, SignScript irá agir como um editor, ou até mesmo uma IDE por ter uma tilização bem especifica que será a programação utilizando linguagem de sinais. O que ela fará exatamente? Será uma tecnologia em computer vision que irá capturar os sinais feitos por alguém em frente a camera e "traduzí-los" em uma linguagem de programação, que será JavaScript, legal, né?
